## 🚀 Deploy to GitHub Pages

```bash
# 1. Create repo
gh repo create ooww-website --public

# 2. Push to GitHub
cd /root/ooww-website
git init
git add .
git commit -m "Initial commit: $OOWW meme coin website"
git branch -M main
git remote add origin https://github.com/harshasoldev/ooww-website.git
git push -u origin main

# 3. Enable GitHub Pages
# Go to repo → Settings → Pages → Source: Deploy from branch (main, /root)
```
